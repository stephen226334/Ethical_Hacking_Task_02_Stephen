OBJECTIVE
_________________________________________________________

The objective of this task is to understand the Network
Scanning and Service Enumeration phase of Ethical
Hacking using the Nmap tool. The task focuses on
identifying open ports, running services, service
versions, and operating system information safely on
an authorized system.

_________________________________________________________
COMMANDS USED
_________________________________________________________
1. nmap --version

Purpose:
Verify that Nmap is installed correctly.

---------------------------------------------------------

2. nmap localhost

Purpose:
Scan localhost and identify open ports.

---------------------------------------------------------

3. nmap -sV localhost

Purpose:
Detect running service versions.

---------------------------------------------------------

4. sudo nmap -O localhost

Purpose:
Identify the operating system using OS fingerprinting.

CONCLUSION
_________________________________________________________
This task provided practical experience with network
scanning and service enumeration using Nmap. I learned
how to identify active hosts, open ports, running

target operating system. Overall, this task improved my
understanding of safe scanning techniques and reinforced
